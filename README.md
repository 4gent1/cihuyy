# OTP SPAM TOOL

Aplikasi spam OTP unlimited untuk testing keamanan sistem OTP.

## Features

- ✅ Spam OTP Unlimited
- ✅ Multi-provider support (7+ providers)
- ✅ Auto-retry mechanism
- ✅ Clean & modern interface
- ✅ Cross-platform (Windows/Linux/Mac)
- ✅ No password required

## Requirements

```bash
pip install requests
```

## Usage

```bash
python spam_otp.py
```

Masukkan nomor target dengan format: `08xxxxxxxxxx`

## Supported Providers

1. Adiraku
2. Singa
3. SpeedCash
4. BisaTopUp
5. JogjaKita
6. Cairin
7. Adiraku V2
